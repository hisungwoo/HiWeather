<h2>:memo: 주요 기능 </h1>

* 현재 내 위치의 날씨, 기온, 습도, 강수확률, 강수량 등 날씨정보 제공

* 오늘의 최고기온, 최저기온, 시간 별 날씨 및 기온 제공

<h2>:computer: 개발 목표 </h1>

* 시스템 아키텍처 설계

* MVP 패턴 적용

* Retrofit을 활용하여 Api 연동

* 앱 출시 

* 출시 후에도 공부한 기술 스택 적용 후 지속적인 업데이트
